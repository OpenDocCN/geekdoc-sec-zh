Appendices

原文：http://www.fuzzingbook.org/html/99_Appendices.html

This part holds notebooks and modules that support other notebooks.

  • Handling Errors provides means to catch and report expected errors within notebooks.

  • Timer is a simple utility to measure how much time elapses during execution.

  • Control Flow helps determining control flow within Python programs.

  • Railroad Diagrams visualize grammars as railroad diagrams.

  • Index is the index of the book.
