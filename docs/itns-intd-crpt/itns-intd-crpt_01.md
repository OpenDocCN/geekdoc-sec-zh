# 密码学密集导论

> 原文：[`intensecrypto.org/public/`](https://intensecrypto.org/public/)

**博阿兹·巴拉克**

*工作正在进行中*

这些是关于密码学入门但节奏快速的本科生/研究生课程的讲义。我正在使用这些讲义为[哈佛 CS 127](https://cs127.boazbarak.org)授课。

您也可以下载[所有讲义的单个 PDF 文件](https://files.boazbarak.org/crypto/lnotes_book.pdf)。

如果您有任何评论、建议、错别字修正等，如果您能在[GitHub 仓库](https://github.com/boazbk/crypto)中发布[问题](https://github.com/boazbk/crypto/issues)或[拉取请求](https://github.com/boazbk/crypto/pulls)，我将不胜感激，我在那里维护这些讲义的源文件。
