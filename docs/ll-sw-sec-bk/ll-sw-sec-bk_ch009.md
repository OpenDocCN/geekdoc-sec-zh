# 附录：贡献指南

如果您想开始为这本书做出贡献：请这么做，我们期待您的贡献！

项目托管在 [`github.com/llsoftsec/llsoftsecbook`](https://github.com/llsoftsec/llsoftsecbook)。我们还有一个 Discord 服务器，您可以在 [`discord.gg/Bm55Z9Ppgn`](https://discord.gg/Bm55Z9Ppgn) 与我们进行互动聊天。

我们使用 [github issues](https://github.com/llsoftsec/llsoftsecbook/issues) 作为我们的问题跟踪器，并使用 [github pull requests](https://github.com/llsoftsec/llsoftsecbook/pulls) 来接受编辑、更改、新增等内容。

如果您想贡献，但不确定从哪里开始，标记为 “[good first issue](https://github.com/llsoftsec/llsoftsecbook/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)” 的开放问题列表可能会给您提供一些贡献灵感的点。请记住，也不要害羞地向我们伸出援手，在 [Discord](https://discord.gg/Bm55Z9Ppgn) 上联系我们。

我们遵循本项目中的 [Contributor Covenant 行为准则](https://github.com/llsoftsec/llsoftsecbook/blob/main/CODE_OF_CONDUCT.md)。

关于如何为本书撰写文本的更多详细信息，请阅读 [contributing.md](https://github.com/llsoftsec/llsoftsecbook/blob/main/contributing.md)。如果在阅读后，您认为某些特定方面可以解释得更好，请通过提出一个 [issue](https://github.com/llsoftsec/llsoftsecbook/issues/new) 来让我们知道。
