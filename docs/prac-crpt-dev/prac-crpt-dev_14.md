## [](#summary)摘要

+   **JavaScript** 和 **Python** 提供简单的加密库

    +   哈希、ECC、ECDSA、AES 以及更多

+   在 **C#** 中，加密较为复杂

    +   使用 **Bouncy Castle .NET** 进行通用加密

    +   或者使用 **Nethereum** 以简化 secp256k1

+   在 **Java** 中，加密较为复杂

    +   **JCA** 和 **Bouncy Castle** 难以使用

    +   **Web3j** 是简化版的 secp256k1 库
